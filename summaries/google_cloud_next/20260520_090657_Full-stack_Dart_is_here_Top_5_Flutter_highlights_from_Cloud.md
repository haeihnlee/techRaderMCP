# Full-stack Dart is here: Top 5 Flutter highlights from Cloud Next '26

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=WTLohOW-cb4
- **요약 일시**: 2026-05-20 09:06:57

## 🔑 핵심 요약
- **Google Cloud Next '26**에서 Flutter 팀이 꼽은 **Top 5 하이라이트** 정리
- **Full-stack Dart** 시대 개막 — `Cloud Functions`의 **Dart 지원**으로 프론트엔드/백엔드 코드·로직 공유 가능
- **Generative UI** 데모(Gen Latte, Gemini + Nano Banana)와 **Toyota·Talabat** 같은 대규모 프로덕션 활용 사례 부각

---

## 📣 주요 발표 내용
- **Full-stack Dart 출시**
  - `Cloud Functions`에서 **Dart 런타임** 공식 지원 → 서버사이드 개발에 Dart 활용 가능
  - 프론트엔드(Flutter)와 백엔드(Cloud Functions) 간 **코드/모델/로직 공유** 가능
- **Generative UI 데모 — "Gen Latte"**
  - **Flutter + Firebase** 앱에서 **Gemini**와 **Nano Banana** 모델로 사용자가 원하는 이미지 협상
  - 생성된 이미지를 실제 라떼 아트로 인쇄해 제공한 부스 데모
- **엔터프라이즈 활용 사례 토크**
  - **Toyota**와 **Talabat**이 Flutter를 **대규모 프로덕션**에서 사용한 사례 발표
- **Developer Keynote**
  - 前 Angular, 現 Flutter 팀 **Emma Twersky**가 키노트에서 Flutter/Firebase **다중 에이전트(many-agent) 데모** 시연
  - 라스베이거스 마라톤 플래닝을 자동화한 멀티 레이어 에이전트 프로젝트 공개
- **Flutter Developer Meetup**
  - 개발자들이 직접 만든 **Generative UI 프로덕션 사례** 다수 공유
- **크로스플랫폼 인지도 확산**
  - 하나의 코드베이스로 **TV·폰·태블릿·PC**까지 동작하는 데모로 Cloud 중심 개발자에게 Flutter 소개

---

## 💡 개발자 포인트
- **Full-stack Dart는 게임 체인저**: Flutter 앱 개발자가 별도 백엔드 언어 없이 `Cloud Functions`까지 Dart 한 언어로 처리 가능 → 모델 클래스/검증 로직 공유 가능
- **Generative UI 패턴**이 더 이상 데모용이 아니라 **실 프로덕션 진입** 단계 — Gemini + Flutter 조합 검토 가치 있음

> ⚠️ Cloud Functions의 Dart 런타임은 새로 추가된 지원이므로 기존 Node.js/Python 함수와의 **상호 운용·배포 파이프라인** 차이를 확인해야 함

- 멀티플랫폼 UI는 **responsive** 설계가 필수 — 같은 코드여도 폼팩터별로 다르게 보이는 것이 의도된 동작
- 키노트·세션 영상 및 블로그 포스트는 영상 설명란 링크에서 후속 공개 예정
