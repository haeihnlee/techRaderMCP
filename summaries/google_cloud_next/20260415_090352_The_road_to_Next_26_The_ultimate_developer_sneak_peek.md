# The road to Next ‘26: The ultimate developer sneak peek

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=KPTW4L-BQO8
- **요약 일시**: 2026-04-15 09:03:52

---

## 🔑 핵심 요약

- **Google Cloud Next 2026**이 약 1주 후 라스베이거스에서 개최 (역대 최대 규모)
- 개발자 전용 키노트 **"Get Real: Agents in the Autonomous Era"** 에서 실제 동작하는 데모 중심의 발표 예정
- **67,000 sq ft** 규모의 쇼케이스 플로어에서 125개 이상의 실습 데모와 체험 제공

---

## 📣 주요 발표 내용

### 🎤 개발자 키노트
- 🤖 테마: **에이전틱(Agentic) AI / 자율 에이전트 시대**
- 🔗 모든 데모가 일관된 스토리라인으로 연결되어 각 제품·런치가 **어떻게 상호작용하는지** 직접 시연
- 🎯 슈트 없는 개발자 친화적 발표 형식

### 🗺️ 쇼케이스 플로어 구성 (3개 Zone)

| Zone | 색상 | 주요 내용 |
|------|------|-----------|
| **Imagine** (우측) | - | AI 데모, 고객 사례, Gemini + 로보틱스 체험 |
| **Learn** (중앙) | 빨강 | 기술 딥다이브, Learn Pod, 데이터 분석·보안·워크스페이스 데모 |
| **Build** (좌측) | - | 핸즈온 워크숍, 에이전틱 해킹존, ADK/A2A 실습 |

### 🛠️ 주목할 데모

- 🚀 **CLI Mission Control**: `Gemini CLI`를 사용해 로켓 발사 시퀀스를 완료하는 리더보드 경쟁 데모
- 🤝 **ADK & A2A in Action**: `ADK`(Agent Development Kit)와 `A2A` 프로토콜로 에이전트가 자율적으로 예산 협상·트렌드 전환·게임 출시까지 수행
- 📱 **Agentic Mobile and Web** (6개 데모):
  - 전화번호 인증
  - **Dart** 풀스택 + **Firebase Functions**
  - 다양한 프레임워크/언어/SDK 지원
  - 오랜 기간 요청받아온 개발자 피처 요청 반영
- ☕ **Gem Latte**: **Generative UI** 기반 커피 주문 앱 — 실제 바리스타 스테이션 운영, 에이전틱 코딩 + 코드 탐색 + 실제 커피 수령

### 🎓 학습·자격증 관련
- **Skills Challenge**: 배지·리워드 적립 게임 (쇼케이스 전 구역 활성화 미션 완료 시 스웨그 제공)
- **3층 Skills Zone**: 핸즈온 워크숍 + 미니 랩
- 파트너 워크숍: **NVIDIA**, **McLaren**, **Team USA** 참여

## 💡 개발자 포인트

- 🔧 `ADK` (Agent Development Kit)와 `A2A` 프로토콜이 이번 Next의 핵심 기술 스택으로 부상
- 📱 **모바일/웹 에이전트** 통합이 주요 방향 — Dart, Firebase, 주요 프레임워크 대상 신규 기능 다수 공개 예정
- 💻 `Gemini CLI` 관련 실습 데모가 공개되므로 CLI 기반 AI 워크플로우 관심 개발자라면 필참

> ⚠️ **주목**: `ADK`와 `A2A` 프로토콜 기반의 **멀티 에이전트 자율 협업** 패러다임이 본격 소개될 예정. 기존 단일 에이전트 아키텍처에서 전환을 고려하는 개발자는 해당 세션 우선 확인 필요.

- 🎯 세션은 600개 이상이지만, 쇼케이스 플로어 자체만으로도 충분한 기술 학습 가능
- 🗣️ 개발자 전용 키노트 외 **Lightning Talk 34개** (각 20분 + Q&A, 데모 중심)도 챙길 것
  - 예: "AI 어시스턴스로 앱 빠르게 빌드하기", "확장 가능한 앱·에이전트·API 빠르게 배포하기"
