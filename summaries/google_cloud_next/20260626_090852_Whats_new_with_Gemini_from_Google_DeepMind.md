## 📣 주요 발표 내용
- **Gemini 3.1**: 멀티모달(영상·텍스트·오디오·이미지·코드) 네이티브 설계, 복잡한 다단계 플래닝 및 툴 사용 등 **에이전트 역량** 대폭 강화
- **Gemma 4**: Gemini 3 계열 파생 오픈소스 모델. **20억~300억 파라미터** 범위, 온디바이스·태스크 특화 학습에 최적, 오디오/비디오 이해 내장
- **Gemini Live**: 오디오 토큰 입출력 네이티브 음성 모델. 저지연·표현력 제어·감정 미러링(**proactive audio**) 지원, Gemini 컨슈머 앱에 배포 완료
- **Lyria 3 Pro**: 텍스트/이미지 프롬프트로 **최대 3분 풀 보컬 곡** 생성
- **Gemini Deep Research 에이전트**: 단일 API 호출로 웹 + 자체 데이터 소스 그라운딩, 텍스트·차트·인포그래픽 생성
- **Genie 3**: 텍스트/이미지 프롬프트로 **인터랙티브 3D 월드** 생성, 키보드로 탐색 (AI Ultra 구독자 대상)
- **Gemini Robotics ER 1.6**: `ER`=Embodied Reasoning. Boston Dynamics **Spot** 로봇에 적용, 객체 카운팅·게이지 판독 등 수행

---

## 💡 개발자 포인트
- **Deep Research가 단일 API 호출로 제공** — 웹 검색 + 자체 데이터 그라운딩 + 차트/인포그래픽 출력을 워크플로 첫 단계로 활용 가능
- `Flash`는 성능·효율 밸런스의 **주력(workhorse) 모델**, `Pro`는 에이전트·코딩·STEM 용, `Flash-Lite`는 최경량·최고속
- **Gemma 4**는 온디바이스 실행·태스크 특화 파인튜닝에 적합한 오픈웨이트 선택지

> Google 제품에 쓰이는 모델과 Cloud에 제공되는 모델이 **동일**하며, 신규 모델은 출시 당일 Cloud 플랫폼에서 바로 사용 가능합니다.
