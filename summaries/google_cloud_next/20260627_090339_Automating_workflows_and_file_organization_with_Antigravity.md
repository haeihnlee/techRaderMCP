# Automating workflows and file organization with Antigravity

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=a7oMylc_SWg
- **요약 일시**: 2026-06-27 09:03:39

---

## 🔑 핵심 요약
- **Antigravity**는 자연어 프롬프트만으로 개인 워크플로우 도구를 만들어주는 AI 코딩 에디터다.
- 발표자는 흩어진 Git 프로젝트·파일을 **태그 기반 타임박스**(today / this week / last month 등)로 정리하는 도구를 직접 생성했다.
- 단순 코드 생성을 넘어 **반복 작업(cognitive toil) 감소**에 초점 — 개발자는 설계와 리뷰에 집중.

---

## 📣 주요 발표 내용
- **프롬프트 → 완성 확장 프로그램**: 텍스트 프롬프트만으로 `Swift` 기반 확장 프로그램 전체를 자동 생성.
- 동일 아이디어를 **Chrome 확장 프로그램**으로도 그대로 재생성(재사용성 강조).
- 핵심 사용 사례는 **파일/프로젝트 정리 자동화** — 시간 단위 태그로 "이번 주/지난주에 뭘 했는지"를 클릭 한 번에 탐색.
- 여러 Git 프로젝트와 파일 시스템에 분산된 작업물을 한곳에서 관리하는 **개인 생산성 워크플로우** 구축에 초점.

---

## 💡 개발자 포인트
- Antigravity는 IDE를 넘어 **워크플로우 빌더**로 동작 — 일회성 코드가 아니라 반복 사용할 개인 도구를 만들어낸다.
- 같은 의도를 `Swift`(네이티브) ↔ Chrome 확장(웹)으로 **타깃 전환 생성**할 수 있어, 멀티 플랫폼 프로토타이핑 비용이 크게 줄어든다.

> 발표의 메시지는 "코드를 더 빨리 짜는 것"이 아니라 **반복적 인지 부담(cognitive toil)을 없애 설계·리뷰에 시간을 쓰게 하는 것**이다. 도구 자동 생성의 결과물 검토(리뷰) 책임은 여전히 개발자에게 있음에 유의.
