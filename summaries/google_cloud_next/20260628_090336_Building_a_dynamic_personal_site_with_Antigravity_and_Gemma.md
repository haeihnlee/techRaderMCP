# Building a dynamic personal site with Antigravity and Gemma 4

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=B41MWQH9qaU
- **요약 일시**: 2026-06-28 09:03:36

---

## 🔑 핵심 요약
- **Antigravity**로 순수 `HTML`/`CSS` 기반 개인 사이트를 제작하고, VS Code 스타일 편집 경험을 그대로 사용
- 포스트별 **추천(related posts)** 기능을 서버 없이 **빌드 타임**에 정적으로 생성
- **Gemma**로 각 글 요약 생성 + **EmbeddingGemma**로 벡터화하여 추천을 사전 계산

---

## 📣 주요 발표 내용
- **Antigravity** 기반으로 개인 웹사이트를 처음부터 제작 (전부 `HTML`/`CSS`)
- VS Code와 유사한 **전체 뷰 편집 문법** 제공 — 실제 VS Code는 아니지만 동일한 syntax로 작업 가능
- **테마 변경** 등 UI 커스터마이징 지원
- 관련 글 추천 기능에 **벡터 데이터베이스** 활용
- 추천 로직 파이프라인:
  - **Gemma**로 각 포스트의 요약문 생성
  - **EmbeddingGemma**로 각 요약을 임베딩 벡터로 변환
  - **빌드 타임 제너레이터**가 사이트를 빌드하면서 각 페이지의 related posts를 사전 생성 후 배포

---

## 💡 개발자 포인트
- 추천 시스템을 위해 **별도 서버를 띄울 필요가 없음** — 빌드 단계에서 모두 사전 계산
- 결과물은 **완전한 정적 사이트**라 배포·운영 비용이 낮음

> 추천 결과가 빌드 시점에 임베딩으로 계산·고정되므로, **오프라인(서비스 연결 없이)** 환경에서도 보고 있는 글 기준으로 관련 글 추천이 동작합니다.

- 온디바이스/경량 모델(`Gemma`, `EmbeddingGemma`) 조합으로 AI 기능을 로컬 파이프라인에 통합하는 사례
- 기존에 `CSS`·엣지 케이스로 막혔던 작업을 AI 코딩 도구로 완성한 워크플로우 예시
