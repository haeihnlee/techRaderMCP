# Build a full stack app with Antigravity voice prompts & sub-agents

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=HA1ORLiuuTc
- **요약 일시**: 2026-06-29 09:03:42

---

## 🔑 핵심 요약
- **음성 프롬프트** 하나로 멀티페이지 풀스택 앱을 **서브 에이전트**들이 분담해 빌드하는 데모
- **Antigravity**의 음성 입력은 코딩 언어·제품명(Gemini 등)을 인식해 일반 STT보다 정확
- 서브 에이전트가 또 다른 서브 에이전트를 띄우는 **오케스트레이션**과 `/goal` 슬래시 커맨드가 핵심

---

## 📣 주요 발표 내용
- **음성 프롬프트 기반 빌드**: 마이크로 "Docker Compose로 노트 앱 만들어줘"를 받아쓰기 → 구현 계획 자동 생성
- **서브 에이전트 분담**: `DevOps and QA engineer` 같은 역할별 에이전트가 API 호출·작업 완수를 자동 수행
- 프론트엔드 / 백엔드 에이전트로 **작업 분리** 가능, 한 에이전트가 전체를 **orchestrate**도 가능
- **다국어 앱** 생성: 영어·힌디어·스페인어·중국어·일본어 동시 지원, 영→스페인어 로컬라이즈 정상 동작
- 백엔드는 **Go** + `SQLite` 저장, 마크다운 작성·프리뷰 및 **URL 공유** 기능 구현
- 빌드에는 **modern web guidance**와 **Chromium** 스킬 활용

---

## 💡 개발자 포인트
- 서브 에이전트에 "프론트엔드 담당", "백엔드 담당" 식으로 **역할(hat)** 을 부여해 병렬 작업 구성
- `/goal` 커맨드로 목표 기반 자율 루프 실행 가능

> **추천**: `/goal`에 "테스트 커버리지 100% 될 때까지 테스트 실행·코드 수정 반복"을 지정하면 목표 달성까지 자동 반복 — 발표자가 여러 번 써봤고 매우 잘 동작한다고 강조

- 에이전트에게 "이 계획 쓸 만한가? 반박해봐"처럼 **push back**을 요청해 계획 품질을 검증
- `modern web guidance`는 아직 설치 안 했다면 꼭 설치 권장
- 입력은 음성 프롬프트 하나뿐, 마지막 "run the app"만 추가 → 사실상 **무인 빌드** 수준
