## 🔑 핵심 요약
- 패션 스타일링 앱 **Whering**(2022년 출시, 글로벌 1,000만 사용자)이 **멀티모달 AI**를 활용해 옷장 디지털화·스타일링을 자동화한 사례
- **Gemini 2.5 Flash Image (`Nano Banana`)** 도입으로 이미지 생성 품질과 **비용 효율**을 동시에 잡아 빠른 기능 출시 가능
- AI 발전으로 비즈니스 모델 자체가 전환: "옷장 전체 디지털화 후 스타일링" → "디지털화 없이도 즉시 가치 제공, 점진적 데이터 축적"

---

## 📣 주요 발표 내용
- Whering의 **AI 활용 3대 축**:
  - **대화형 채팅(Conversational Chat)**: 개인 스타일 변화 인사이트, 리셀 추천 가격, 룩 업그레이드 제안
  - **옷장 디지털화 자동화**: 카메라롤·Instagram에서 직접 스캔하는 **Gallery Scanning** 기능으로 진입 장벽 제거
  - **체험형 기능**: **가상 피팅(Virtual Try-on)**, **컬러 분석**, 기분·날씨·목적지 기반 스타일링
- **컴퓨터 비전**으로 업로드 시 아이템 자동 **크롭·태깅**, 이미지 향상(enhance) 기능 출시 예정
- AI를 **전면 노출**(NAV 바의 채팅)과 **백그라운드 처리**(자동 크롭/태깅)로 분리해 사용자층별 경험 최적화

---

## 💡 개발자 포인트
- **비용 중심 모델 선택 전략**: Gen 미디어(생성형 이미지) 모델은 비용 부담이 크므로, COG(원가)가 높은 AI 기능은 **유료 페이월(paywall) 뒤로 게이팅**
- 향후 **음성(Voice)** 기능을 비용 출력 관점에서 우선 검토 중

> 핵심 인사이트: AI가 빠르게 발전하면 기존에 구축한 비즈니스 로직과 기능 세트 자체가 더 이상 필요 없어질 수 있다. "디지털화 선행" 같은 무거운 전제를 걷어내고, 더 가벼운 진입점(셀카 로깅 등)으로 즉시 가치를 제공하는 방향으로 모델을 재설계.

- 모델 선택·테스트 기준: ① 10만+ 메시지 규모의 **사용자 리서치 백로그** 기반 전략적 우선순위화 ② 컬처/트렌드(zeitgeist) 부합 여부 ③ 제품 비전과의 정합성
- UX 철학 "Wardrobe Zen": 스타일링 앱답게 **클린 UI**와 안전한 실험 공간을 의도적으로 설계 — 커뮤니티·브랜드를 경쟁 해자(moat)로 인식
