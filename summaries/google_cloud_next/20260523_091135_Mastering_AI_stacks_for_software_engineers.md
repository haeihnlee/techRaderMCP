## 📣 주요 발표 내용
- **콘텐츠 방향성**: 트렌드보다 *자신의 호기심*을 따르는 **niche 전략**이 더 큰 성장을 만듦
- 가장 자랑스러운 영상은 **Inference Engine** 해설 — 토큰 생성 속도와 추론 품질이 어떻게 결정되는지 first principle로 풀어냄
- **로컬 LLM 실행 트렌드**: Apple Silicon·GPU를 구매해 집에서 모델을 돌리는 개발자가 늘면서, 하드웨어 병목을 이해하기 위한 inference 지식 수요 급증
- Jensen Huang이 자주 언급하는 **5-layer cake**(Application / Model / Infra / Chip / Energy)로 AI 스택을 체계적으로 분석
- 영상 편집은 100% 본인이 직접 수행, 협업·계약 관리는 매니저(Ja'Marr, Amy)에게 위임하여 콘텐츠 제작에 집중

## 💡 개발자 포인트
> **Sonnet 3.5 출시는 개발자 직무 변화의 변곡점이었다.** AI 도구의 동작 원리를 모르면 단순 사용자에 그치며, Coding Agent를 제대로 활용하려면 한 단계 아래(Inference) 이해가 필수.

- **로컬 LLM 실행을 고려한다면**: 어떤 모델을 돌릴 수 있는지, 어느 정도 성능이 나오는지는 **inference 엔진의 동작 원리**를 알아야 판단 가능
- **Coding Agent**를 로컬에서 실행하는 LLM과 페어링하는 트렌드가 확산 중 → 클라우드 의존도를 낮춘 개발 환경 구성 가능
- 콘텐츠 제작자로 전환하려는 개발자에게: **구독자 수보다 niche에 대한 신뢰**가 핵심. 작은 커뮤니티라도 가치를 주면 기회는 따라옴
- 과거 실패한 채널 경험도 편집·기획 스킬로 **그대로 누적**된다는 점 — 사이드 프로젝트의 학습은 결코 낭비가 아님
