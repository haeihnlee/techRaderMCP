# How @CodeWithHarry helps developers upskill with AI

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=iPzaKfSq-Q4
- **요약 일시**: 2026-05-23 09:08:48

---

## 🔑 핵심 요약
- **CodeWithHarry** (구독자 9.6M)가 Google Cloud Next에서 **AI 시대 개발자 업스킬링** 전략을 공유
- 핵심 메시지: **AI/에이전트를 대체자가 아닌 도구로 활용**하고 꾸준히 학습하라
- Google **Anti-gravity** + **Gemini 3.5 Flash** 조합과 **Google Glass**에 대한 기대감 표명

---

## 📣 주요 발표 내용
- **채널 성장 비결**: 기초(basics)부터 시작해 점진적으로 깊이(depth)를 쌓는 콘텐츠 구조 — "start from basics and raise the bar"
- **언어 장벽 돌파**: 영어뿐 아니라 **힌디어**로도 콘텐츠 제작 → 로컬 언어 시장에서 성공 입증
- **2020년 Project Jarvis**: LLM이 대중화되기 전 음성 기반 에이전트를 자체 구현 ("Hey Jarvis, open Google Chrome")
- **AI 자동화 사례**: 카메라 영상 + 폰 마이크 오디오 **자동 싱크 Python 스크립트** 직접 작성 → 현재는 **Google AI Studio** 단일 프롬프트로 동일 앱 생성 가능
- 현재 진행 중인 사이드 프로젝트: **AI 터미널** (Claude Code / Gemini CLI / Codex 스타일, 모델 플러그인 방식)

---

## 💡 개발자 포인트
- **마인드셋**: AI를 도구로 활용하면 커리어 성장 가능 — "AI 때문에 코딩 배우지 말자"는 핑계 금지
- **콘텐츠 제작 진입 장벽 하락**: 과거 개발자만 가능했던 자동화 작업이 이제 **AI Studio 프롬프트 한 줄**로 해결됨

> ⚠️ **창의성과 인간의 사고는 대체 불가능** — "COBOL 시대부터 개발자 대체 얘기는 있었지만 아직 일어나지 않았다"

- **유튜브 시작 팁**: 자신의 언어·커뮤니티를 타겟으로 작게 시작하라. 10M 구독자와 비교하지 말고 자신만의 청중에게 집중
- **추천 학습 경로**: 자료구조/알고리즘 → Java 기초 → 웹 개발 (CodeWithHarry 무료 플레이리스트 활용 가능)
