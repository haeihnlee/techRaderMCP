## 🔑 핵심 요약
- Google Cloud의 **Agent Development Kit(`ADK`)** 을 활용해 **텍스트 어드벤처 게임**을 공략하는 라이브 코딩 세션
- AI 에이전트가 자연어 게임 환경을 **인식 → 추론 → 행동** 하는 루프를 실제 코드로 시연
- 발표자: **Tilde Thurium**, **Abdel Sghiouar** (Google Cloud Developer Advocate)

> ⚠️ 이 영상은 라이브스트림(2026-06-17 9:30 AM PDT 방송 예정)으로, 현재 **자막/본문이 제공되지 않아** 영상 설명 메타데이터 기반으로 요약했습니다. 정식 내용은 방송 후 자막 확보 시 갱신이 필요합니다.

## 📣 주요 발표 내용
- **`ADK`(Agent Development Kit)** 를 메인 도구로 사용한 라이브 코딩 데모
- 텍스트 어드벤처 게임을 **에이전트가 직접 플레이/클리어** 하는 과제 설정
- 게임 상태 텍스트를 입력받아 LLM 기반 에이전트가 다음 행동(명령어)을 생성하는 구조 시연 예상

---

## 💡 개발자 포인트
- **`ADK`** 는 Google Cloud의 에이전트 개발 프레임워크로, 도구 호출·상태 관리·추론 루프를 코드로 구성 가능
- 텍스트 어드벤처 같은 **상태 기반 환경**은 에이전트의 계획(planning)·메모리 설계를 학습하기 좋은 예제
- 라이브 코딩 포맷이므로 실제 **에이전트 구현 패턴**과 디버깅 흐름을 참고하기 적합

> 현재 메타데이터만으로는 구체적 API·코드 디테일을 확인할 수 없으므로, 실 적용 전 공식 ADK 문서 확인 권장
