# Build an app from your phone in AI Studio

- **컨퍼런스**: Google Cloud Next
- **출처**: https://www.youtube.com/watch?v=lKUohuLy43M
- **요약 일시**: 2026-06-10 09:04:26

---

## 🔑 핵심 요약
- **AI Studio**에서 앱 설치 없이 **모바일 브라우저만으로** 앱을 "바이브 코딩"하는 데모
- 자연어(텍스트 또는 음성)로 요청하면 AI가 **완전한 React 앱**을 자동 생성
- 이미지 생성은 **Nano Banana API**를 호출해 처리 (예: 커스텀 봉제인형 생성기)

---

## 📣 주요 발표 내용
- 브라우저에서 `aistudio.google.com` 접속 → **별도 앱 설치 불필요**
- 만들고 싶은 앱을 **텍스트 입력** 또는 **마이크 음성**으로 설명하면 됨
- AI가 사용자의 지시를 자동으로 **정제·최적화(optimize)** 한 뒤 AI Studio로 전달
- **chat 아이콘**으로 AI의 사고 과정(reasoning)과 생성 중인 코드 파일 실시간 확인 가능
- 백엔드에서 **풀스택 React 앱**을 생성하며, `index.css` 등 스타일링도 자동 결정
- 데모 앱 "Julia's Plushy Palace": 동물·색상·액세서리 선택 → `Nano Banana API`로 이미지 생성
- 데스크톱(노트북)에서도 동일하게 동작하며, **코드 직접 편집**도 가능

---

## 💡 개발자 포인트
- 모바일 환경에서도 **앱 설치 없이 풀스택 프로토타이핑**이 가능 → 빠른 아이디어 검증에 유용
- AI가 입력 프롬프트를 자동 정제해주므로, 거친 자연어 요청도 동작하는 앱으로 변환
- 이미지 생성을 위해서는 **AI Studio API 키** 발급이 필요

> 데모 앱은 최초 실행 시 사용자가 직접 **AI Studio API 키를 선택**하도록 설계됨 — `Nano Banana API` 호출에 이 키가 사용되므로 키 발급/관리가 전제 조건

- 생성 결과물은 편집 가능한 **실제 코드**이므로, 프로토타입을 그대로 커스터마이징해 발전시킬 수 있음
