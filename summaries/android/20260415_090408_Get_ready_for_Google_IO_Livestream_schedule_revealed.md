# Get ready for Google I/O: Livestream schedule revealed

- **컨퍼런스**: Android
- **출처**: https://android-developers.googleblog.com/2026/04/get-ready-for-google-io-livestream-schedule-revealed.html
- **요약 일시**: 2026-04-15 09:04:08

---

## 🔑 핵심 요약
- **Google I/O 2026**이 **5월 19~20일** 온라인 라이브스트림으로 개최
- **AI**, **Android**, **Chrome**, **Cloud** 전반의 최신 업데이트 발표 예정
- **에이전틱(Agentic) 코딩** 및 자율 워크플로우 중심의 개발자 도구 진화가 핵심 테마

---

## 📣 주요 발표 내용

- 🤖 **에이전틱 개발 시대**: AI가 복잡한 작업을 자율 처리하는 **Agentic Workflow** 도구 공개
- 📱 **Android 개발 혁신**: 프로토타이핑부터 네이티브 폴리시까지 전 과정을 AI로 가속화
- 🌐 **에이전트 지원 웹 앱 빌드**: 브라우저 내 인터랙티브 UI 배포, 복잡한 디버깅 자동화 도구 소개
- 🎤 **Google 키노트**: `10:00 AM PT` (5월 19일)
- 👩‍💻 **Developer 키노트**: `1:30 PM PT` (5월 19일)
- 🎬 주요 세션 예고:
  - `What's new in Android`
  - `What's new in Google AI`
  - `What's new in Chrome`
  - `Build core skills to thrive as an AI-era developer`

## 💡 개발자 포인트

- **Agentic Coding** 패러다임으로 전환 — AI가 단순 보조가 아닌 **자율 실행 주체**로 동작하는 워크플로우 설계 필요
- **Android 앱 개발 파이프라인** 전반에 AI 통합이 강화될 예정이므로, 새로운 툴체인 학습 대비 권장
- 웹 개발자는 **에이전트 지원 웹 애플리케이션** 아키텍처 변화에 주목 필요

> ⚠️ **주의**: 5월 21일에 온디맨드 세션과 **코드랩(Codelabs)** 이 별도 공개됩니다. 라이브 세션 이후에도 추가 학습 자료가 제공되므로 일정을 미리 확보해 두세요.

- 사전 등록 시 전체 프로그램 탐색 및 최신 업데이트 알림 수신 가능 → [Register](https://io.google) 권장
