# Android Developers Blog: Build for the future with the Android XR Developer Catalyst Program — Apply now!

- **컨퍼런스**: Android
- **출처**: https://android-developers.googleblog.com/2026/05/apply-android-xr-developer-catalyst.html
- **요약 일시**: 2026-05-20 09:15:47

---

## 🔑 핵심 요약
- Google이 **Android XR Developer Catalyst Program** 신청을 오픈, 향후 1년 내 출시 예정인 Android XR 앱 개발자를 지원
- **유선 XR 글래스**(XREAL Project Aura)와 **인텔리전트 아이웨어**(오디오/디스플레이 글래스) 대상 개발 키트·기술 지원·**보조금(non-recoupable grant)** 제공
- 신청 마감: **2026년 6월 30일 23:59 PDT**

---

## 📣 주요 발표 내용
- **프로그램 명칭**: Android XR Developer Catalyst Program
- **지원 항목 3가지**:
  - **Development Kits**: `XREAL Project Aura`(유선 XR 글래스) 및 인텔리전트 아이웨어 하드웨어 개발 키트 조기 접근권
  - **Technical Support**: Google Play 출시 준비를 위한 전용 기술 리소스·지원 포럼
  - **Grant Opportunities**: 신청 시 **상환 의무 없는 보조금** 수령 가능
- **관심 분야**: 미디어, 게임, 생산성, 헬스 + 기타 혁신적 유스케이스
- **지원 SDK/엔진**: `Kotlin` + **Jetpack XR SDK**, `Unity`, `Unreal Engine`, `Godot`
- 기존 XR Experiments / Samples 참고 가능 (spatial music, navigation 등)

---

## 💡 개발자 포인트
- **대상 자격**: 향후 **6~12개월 내** Android XR 생태계에 앱을 출시하려는 개발자
- Kotlin 외에 **Unity·Unreal·Godot 멀티 엔진을 공식 지원**하므로 기존 게임/3D 개발자도 진입 부담이 낮음
- 하드웨어 입수가 XR 개발의 진입장벽이었으나, 본 프로그램으로 **양산 전 디바이스 조기 확보** 가능

> ⚠️ **신청 마감일 엄수**: `2026-06-30 23:59 PDT`까지 application 제출 필요. 마감 후에는 본 사이클 지원 불가.
