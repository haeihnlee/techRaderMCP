## 📣 주요 발표 내용
- **GenUI(Generative UI)** — LLM 응답에 따라 정적 위젯이 아닌 **UI 컴포넌트 자체를 생성**해 렌더링하는 접근.
- 언어 학습(핀란드어) 시나리오에 GenUI 를 결합해, 학습 콘텐츠에 맞춰 **적응형 UI** 를 만드는 데모.
- `Observable Flutter` 라이브 코딩 시리즈의 일환으로 진행되는 실시간 구현 토론.

## 💡 개발자 포인트
- GenUI 는 **AI 응답 → UI 위젯 매핑** 레이어가 핵심이므로, LLM 출력 스키마와 Flutter 위젯 빌더 간 계약(contract) 설계가 중요합니다.
- 동적 UI 생성 시 **상태 관리·렌더링 성능·예외 처리**(잘못된/예상치 못한 LLM 출력)를 고려해야 합니다.
