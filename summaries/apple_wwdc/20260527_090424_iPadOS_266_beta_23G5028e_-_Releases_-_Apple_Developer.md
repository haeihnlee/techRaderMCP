# iPadOS 26.6 beta (23G5028e) - Releases - Apple Developer

- **컨퍼런스**: Apple WWDC
- **출처**: https://developer.apple.com/news/releases/?id=05262026b
- **요약 일시**: 2026-05-27 09:04:24

---

## 🔑 핵심 요약
- Apple이 **`iPadOS 26.6 beta`** (빌드 `23G5028e`)를 개발자 대상으로 릴리스
- 릴리스 일자: **2026년 5월 26일**
- 다운로드 및 릴리스 노트는 Apple Developer 포털에서 확인 가능

---

## 📣 주요 발표 내용
- **`iPadOS 26.6`** 의 베타 시드 배포 (빌드 번호: **`23G5028e`**)
- 정식 출시 전 개발자 검증을 위한 **developer beta** 단계
- 페이지 상에서는 상세 변경 내역(릴리스 노트) 링크만 제공되며, 본문에 신기능 설명은 포함되지 않음
- 다운로드는 **Apple Developer 계정 로그인** 필요

---

## 💡 개발자 포인트
> 베타 빌드는 **프로덕션 디바이스에 설치하지 말 것** — 데이터 손실·앱 호환성 문제가 발생할 수 있습니다.

- 자사 앱이 **iPadOS 26.6** 정식 출시 전에 호환성을 점검해야 한다면 이 베타 시드로 회귀 테스트 진행 권장
- `Xcode` 의 **iPadOS 26.6 SDK** 와 함께 사용 시 새 API·동작 변경 검증 가능
- 버그 발견 시 **Feedback Assistant** 로 리포트 → 정식 릴리스 전 수정 가능성 향상
- 상세 API 변경점은 페이지 내 **release notes** 링크를 별도로 확인 필요 (이 페이지 본문에는 미포함)
