## 🔑 핵심 요약
- **`iOS 26.5.1`** (빌드 `23F81`)가 **2026년 6월 1일** 정식 릴리스되었습니다.
- 마이너 점(point) 업데이트로, **버그 수정·안정성 개선** 중심의 패치 릴리스입니다.
- Apple Developer 사이트에서 **다운로드** 및 **릴리스 노트** 확인이 가능합니다.

## 📣 주요 발표 내용
- **`iOS 26.5.1`** 정식 버전 배포 — 빌드 번호 **`23F81`**
- 정식 GA(General Availability) 채널로 제공 (베타 아님)
- 페이지 내 제공 항목
  - **View downloads** — SDK·이미지 다운로드 링크

## 💡 개발자 포인트
- `.5.1` 형태의 **3rd-level 패치**이므로, 새로운 API 추가보다는 **회귀 버그 수정·보안 패치** 가능성이 높습니다.
- 앱 호환성 검증 시 **빌드 `23F81`** 기준으로 실기기/시뮬레이터 테스트를 권장합니다.

> ⚠️ 정확한 Breaking Change 여부와 deprecated API 변동은 본문만으로 판단할 수 없습니다. 반드시 **공식 릴리스 노트**를 확인하세요.
