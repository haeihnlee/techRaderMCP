## 💡 개발자 포인트

- 베타 3 단계이므로 **API 변경 가능성**이 여전히 존재함
- 앱의 **iOS 26.5 호환성 테스트**를 이 빌드 기준으로 진행 권장
- 릴리즈 노트(`Release Notes`)에서 **Deprecation**, **Breaking Change**, 신규 API 여부를 반드시 확인할 것

> ⚠️ **Breaking Change 주의**: 베타 빌드 간 API 동작이 변경될 수 있으므로, beta 2 대비 달라진 동작이 있는지 릴리즈 노트를 통해 검토 필수

- **TestFlight** 배포 전 이 베타 빌드로 내부 QA 사이클 재점검 권장
- 빌드 번호 `23F5059e`를 기준으로 **버그 리포트(Feedback Assistant)** 제출 시 명시할 것
