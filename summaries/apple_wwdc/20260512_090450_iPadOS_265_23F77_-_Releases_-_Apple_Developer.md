## 💡 개발자 포인트
> **공식 SDK / Xcode 정합성 확인 필수** — iPadOS 26.5 정식 빌드 기준으로 동작 검증 후 앱 제출 권장.

- 진행 중인 앱은 **iPadOS 26.5 (23F77)** 실기에서 회귀 테스트 수행

- RC2 → 정식 승격이므로 RC2에서 검증을 마쳤다면 추가 변경 가능성은 낮음
- 릴리스 노트에서 **Deprecated API / Known Issues** 항목을 확인하여 영향도 점검
