## 🔑 핵심 요약
- Apple이 **`visionOS 26.6 beta 2`** (빌드 `23O5743c`)를 **2026년 6월 15일** 개발자에게 배포
- Apple Vision Pro 대상 **26.6 정식 출시 전 두 번째 베타**로, 안정화·버그 수정 중심의 점진적 업데이트
- 개발자 계정에서 **다운로드(View downloads)** 및 **릴리스 노트(View release notes)** 확인 가능

---

## 📣 주요 발표 내용
- **`visionOS 26.6 beta 2`** 시드 배포 — 빌드 번호 `23O5743c`
- 배포일: **2026년 6월 15일**
- 베타 1에 이은 **두 번째 베타 시드**로, 26.6 사이클의 안정화 단계 진입
- 상세 변경사항은 Apple Developer 사이트의 **릴리스 노트** 페이지에 별도 게시

---

## 💡 개발자 포인트
- visionOS 앱 개발자는 신규 베타에서 **기존 앱의 호환성·회귀(regression) 테스트**를 권장
- 베타 SDK는 **Xcode를 통해 설치**하며, 디바이스 OTA 또는 복원 이미지로 업데이트

> ⚠️ 베타 빌드는 프로덕션 환경에 사용하지 말고, 별도 테스트 디바이스에서 검증하세요. 정식 릴리스 전까지 API 동작이 변경될 수 있습니다.

> 본 페이지는 릴리스 인덱스 항목으로 세부 변경 내역이 포함되어 있지 않습니다. 구체적인 API 변경·수정 사항은 공식 **릴리스 노트**를 직접 확인해야 합니다.
