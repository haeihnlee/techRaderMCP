## 📣 주요 발표 내용
- `macOS 27.0` 의 **beta 2** 시드 공개 — 빌드 번호 `26A5368g`.
- 베타 2는 일반적으로 신규 API 추가보다는 **beta 1에서 보고된 이슈 수정과 SDK 안정화**가 중심입니다.
- 최신 변경 사항·알려진 이슈(Known Issues)는 Apple Developer의 **릴리스 노트** 페이지에서 직접 확인 필요.

> ⚠️ 베타 SDK 및 OS는 **운영 환경(프로덕션) 기기에 설치하지 말 것**. 별도 테스트 기기 또는 가상화 환경 사용을 권장합니다.

---

## 💡 개발자 포인트
- `macOS 27` 대상 앱을 빌드·테스트하려면 대응되는 **Xcode 27 베타**가 필요합니다.
- beta 1에서 작성한 코드가 beta 2에서 동작이 달라질 수 있으므로, **회귀 테스트(regression test)** 를 다시 수행하세요.

> 📌 **Breaking Change 확인 경로**: 새 동작 변경·Deprecated API는 페이지의 **릴리스 노트**에 정리되어 있으므로, 베타 업데이트 시마다 릴리스 노트를 반드시 확인하는 것을 권장합니다.
