## 💡 개발자 포인트
- macOS 26.6은 마이너 업데이트이므로 일반적으로 **API breaking change 가능성은 낮음**. 다만 베타 SDK 변경 사항이 있을 수 있으므로 Release Notes 확인 권장
- **Xcode beta** 및 **Command Line Tools** 호환성 매트릭스를 함께 점검할 것
- CI/CD 환경에서 `xcodebuild` 사용 시 베타 macOS와 정식 Xcode 조합은 권장되지 않음

> ⚠️ **주의**: 베타 OS는 프로덕션/메인 개발 머신에 설치하지 말고, **별도 파티션 또는 보조 머신**에서 테스트하세요. 데이터 손실·서드파티 호환성 문제가 발생할 수 있습니다.
