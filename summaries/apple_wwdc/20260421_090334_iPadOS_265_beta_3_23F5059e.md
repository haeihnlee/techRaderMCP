## 💡 개발자 포인트

- 🛠️ **beta 3** 단계이므로 프로덕션 배포보다는 **테스트 환경** 적용 권장

> ⚡ **Breaking Change 주의**
> 베타 버전은 빌드마다 API 시그니처나 동작이 변경될 수 있으므로, `beta 3` 기준으로 앱 호환성 재검증을 권장합니다.

- 📲 **TestFlight** 또는 **Developer Profile** 을 통해 실기기 테스트 가능
- 🧪 릴리스 노트의 **Known Issues** 섹션을 통해 현재 알려진 버그 사전 파악 권장
