## 💡 개발자 포인트

- 🔧 **베타 빌드**이므로 프로덕션 배포보다는 **테스트 환경**에서 검증 권장
- 📋 `23F5069b` 빌드 기준으로 앱 호환성 및 API 동작 변경 여부를 **릴리스 노트**에서 반드시 확인 필요

> ⚠️ **Breaking Change 가능성:** 베타 4 단계에서도 API 변경이 포함될 수 있으므로, 이전 베타(`beta 3`)와의 차이점을 릴리스 노트에서 반드시 비교 검토할 것

- 🧪 **TestFlight** 또는 개발자 기기에 설치하여 앱 동작 사전 검증 권장
- 📌 정식 출시 전 최종 대응을 위해 **beta 4 피드백** 제출 창구 적극 활용 권장
