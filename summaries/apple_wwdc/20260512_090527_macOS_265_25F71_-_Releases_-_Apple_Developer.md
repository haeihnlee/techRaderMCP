## 💡 개발자 포인트
> macOS 26.5는 **마이너 업데이트(.5)** 이므로 신규 API보다 안정성·보안 패치 중심일 가능성이 높음. 다만 `Xcode 26.5` 와 SDK 정합성을 맞추려면 **빌드 환경 업그레이드**를 같이 검토해야 함.

- 앱 빌드/배포 머신은 **`25F71`** 빌드로 업데이트 후 회귀 테스트 권장
- **공증(Notarization)** · **App Store Connect** 업로드 도구가 macOS 버전에 의존하는 경우 사전 검증 필요
- Mac Catalyst·SwiftUI·AppKit 앱은 새 빌드에서 **레이아웃/접근성 동작 변화**가 없는지 스모크 테스트 권장
- 정확한 변경 항목은 반드시 **Release Notes** 원문을 확인할 것 (요약 본문에 세부 내용 미공개)
