# App Store Connect Update

- **컨퍼런스**: Apple WWDC
- **출처**: https://developer.apple.com/news/releases/?id=04162026b

## 🔑 핵심 요약
- **App Store Connect**가 2026년 4월 16일자로 업데이트됨

- 개발자 도구 및 앱 관리 워크플로우에 영향을 줄 수 있는 업데이트

## 💡 개발자 포인트

> **주의:** 현재 제공된 정보만으로는 구체적인 Breaking Change나 API 변경 여부를 확인할 수 없습니다.
> 반드시 공식 릴리즈 노트를 확인하여 영향도를 직접 파악하세요.

- **App Store Connect** 업데이트는 아래 영역에 영향을 줄 수 있음
  - 앱 등록 / 심사 제출 프로세스
  - `App Store Connect API` 엔드포인트 변경
  - **TestFlight**, 인앱결제, 구독 관리 기능
  - **Analytics**, 매출 리포트 UI 변경 가능성
- 업데이트 후 기존 자동화 스크립트(`altool`, `xcrun`, API 호출 등) 정상 동작 여부 점검 권장
