## 💡 개발자 포인트
- 베타 3 단계이므로 초기 베타 대비 **버그 수정 및 안정화**가 진행된 빌드일 가능성이 높습니다.
- 동일 시기 배포되는 **`Xcode 26` 베타**로 빌드·테스트해야 최신 SDK를 사용할 수 있습니다.

> 🚨 **주의:** 베타 SDK로 빌드한 앱을 App Store에 정식 제출하지 마세요. 정식(GM/RC) 출시 전까지는 내부 테스트 용도로만 사용해야 합니다.

- 기존 `iPadOS 26` 대상 앱은 이번 베타에서 **회귀(regression)** 여부를 재검증하는 것이 안전합니다.
- 구체적인 **Deprecated API / Breaking Change** 항목은 `View release notes`에서 반드시 확인하세요.
