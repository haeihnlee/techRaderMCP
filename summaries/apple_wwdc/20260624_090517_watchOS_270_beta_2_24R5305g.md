# watchOS 27.0 beta 2 (24R5305g)

- **컨퍼런스**: Apple WWDC
- **출처**: https://developer.apple.com/news/releases/?id=06232026a
- **요약 일시**: 2026-06-24 09:05:17

---

## 🔑 핵심 요약
- Apple이 **`watchOS 27.0 beta 2`** (빌드 `24R5305g`)를 개발자에게 배포했습니다.
- 배포일은 **2026년 6월 23일**이며, WWDC 2026 사이클의 두 번째 베타입니다.
- Apple Developer 페이지에서 **다운로드(View downloads)** 와 **릴리스 노트(View release notes)** 를 확인할 수 있습니다.

---

## 📣 주요 발표 내용
- **`watchOS 27.0`** 의 두 번째 개발자 베타(beta 2)가 공개되었습니다.
- 빌드 번호: **`24R5305g`**
- 개발자는 Apple Developer 포털에서 OTA 또는 이미지 형태로 베타를 받을 수 있습니다.

> 이번 릴리스 페이지에는 상세 변경 내역이 본문에 노출되지 않습니다. 구체적인 API 변경·버그 수정 사항은 페이지 내 **릴리스 노트(View release notes)** 링크에서 직접 확인해야 합니다.

---

## 💡 개발자 포인트
- watchOS 앱을 개발 중이라면 **`watchOS 27 SDK`** 기준으로 빌드·테스트를 진행할 시점입니다.

> ⚠️ **베타 빌드**는 정식 출시 전 단계이므로, 프로덕션 기기·메인 개발 기기에는 설치를 피하고 별도 테스트 기기에 설치하는 것을 권장합니다.

- beta 1 대비 변경 사항(Deprecated API, 동작 변경 등)이 있을 수 있으니 **릴리스 노트(View release notes)** 를 반드시 확인하세요.
