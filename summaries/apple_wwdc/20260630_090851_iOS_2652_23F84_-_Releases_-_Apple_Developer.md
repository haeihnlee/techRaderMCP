# iOS 26.5.2 (23F84) - Releases - Apple Developer

- **컨퍼런스**: Apple WWDC
- **출처**: https://developer.apple.com/news/releases/?id=06292026a
- **요약 일시**: 2026-06-30 09:08:51

---

## 🔑 핵심 요약
- Apple이 **`iOS 26.5.2` (빌드 `23F84`)** 정식 릴리스를 배포했습니다.
- 릴리스 일자는 **2026년 6월 29일**이며, Apple Developer 릴리스 페이지에서 다운로드 가능합니다.
- 정규 버전(`x.x.2`) 패치로, 통상 **안정성·보안 수정** 중심의 마이너 업데이트입니다.

---

## 📣 주요 발표 내용
- `iOS 26.5.2` 빌드 `23F84` 공개 — Apple Developer **Releases** 채널에 등록.
- 페이지에서 **`View downloads`** 링크를 통해 SDK / 디바이스 이미지 다운로드 제공.
- 별도의 신규 API·기능 공지 없이 빌드 정보만 게시된 **점진 패치 릴리스**.

> ⚠️ 이 릴리스 노트 페이지에는 상세 변경 내역(release notes)이 포함되어 있지 않습니다. 구체적 수정 항목은 Xcode의 **What's New / Release Notes** 또는 별도 보안 업데이트 문서에서 확인이 필요합니다.

## 💡 개발자 포인트
- 마이너 패치(`.2`)인 만큼 **API 변경 가능성은 낮으나**, 최신 디바이스 검증을 위해 테스트 디바이스를 `23F84`로 업데이트하는 것을 권장합니다.
- 빌드 번호 `23F84`를 **CI / 디바이스 매트릭스**에 반영해 회귀 테스트 기준선을 갱신하세요.

> 보안 패치가 포함된 경우 사용자 영향이 클 수 있으므로, 앱의 최소 지원 버전 정책과 무관하게 **실기기 검증 대상에 즉시 포함**하는 것이 안전합니다.
