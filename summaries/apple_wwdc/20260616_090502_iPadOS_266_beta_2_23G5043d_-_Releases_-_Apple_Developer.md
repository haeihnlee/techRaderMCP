## 🔑 핵심 요약
- Apple이 **`iPadOS 26.6` beta 2** (빌드 `23G5043d`)를 개발자에게 배포했습니다.
- 배포일은 **2026년 6월 15일**로, `iPadOS 26.6` 정식 출시 전 두 번째 베타입니다.
- 다운로드(`View downloads`)와 릴리스 노트(`View release notes`)가 Apple Developer 사이트에 함께 제공됩니다.

---

## 📣 주요 발표 내용
- **`iPadOS 26.6` beta 2** 시드 공개 — 빌드 번호 `23G5043d`
- 개발자 계정으로 **OTA 업데이트** 또는 구성 프로필을 통한 설치 가능
- 변경 세부 사항은 Apple의 **릴리스 노트** 페이지에서 확인 필요 (본 페이지는 배포 공지만 포함)

> ⚠️ 본 추출 콘텐츠에는 신규 API·기능 상세가 포함되어 있지 않습니다. 정확한 변경점은 `View release notes` 링크의 SDK 릴리스 노트를 직접 확인하세요.

---

## 💡 개발자 포인트
- `26.6` 마이너 버전 베타이므로 **대규모 API 변경보다 안정화·버그 픽스 위주**일 가능성이 높습니다.
- 앱 호환성 검증 시 베타 빌드 `23G5043d`를 **테스트 디바이스에만** 설치하는 것을 권장합니다.

> 베타 OS는 **다운그레이드가 제한적**이므로 메인 작업용 iPad에는 설치하지 마세요.
