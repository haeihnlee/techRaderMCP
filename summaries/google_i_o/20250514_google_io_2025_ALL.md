# 👨‍💻 개발자 키노트

**[Google I/O '25 Developer Keynote](https://youtu.be/GjvgtwSOCao)**

## 핵심 요약 (3줄)
- Gemini 2.5 Pro/Flash 기반의 새로운 개발 도구들(Stitch, AI Studio 코드 에디터, MCP 지원 등)이 대거 공개되어 프롬프트에서 UI 디자인과 코드까지 빠르게 생성 가능해졌다
- Live API에 네이티브 오디오 모델, URL Context, 비동기 함수 호출 등 에이전트 개발에 필수적인 기능들이 신규 추가되었다
- Google Genai SDK에서 MCP(Model Context Protocol)를 네이티브로 지원하게 되어 오픈소스 툴 생태계와의 통합이 대폭 간소화되었다

---

## 주요 발표 내용

### 🎨 Stitch (Google Labs 실험적 프로젝트)
- 프롬프트 → UI 디자인 → 코드 생성까지 원스톱으로 처리
- Gemini 2.5 Pro/Flash 기반으로 디자인 리믹스 및 실시간 수정 지원
- 생성된 결과물은 정적 스크린샷이 아닌 실제 마크업 코드로 제공
- Figma로 내보내기 및 IDE 복사 지원 · 접속: `labs.google/stitch`

### 🔊 Live API 업데이트
- **2.5 Flash 네이티브 오디오 모델** 신규 지원, 24개 언어 네이티브 지원
- **Proactive Audio**: 무관한 소음/음성 자동 무시
- **슬라이딩 컨텍스트 윈도우**: 장시간 세션 유지 가능
- **비동기 함수 호출(Non-blocking)**: MCP 서버 호출 등 고지연 작업을 대화 흐름을 끊지 않고 처리

### 🔗 URL Context 도구
- Gemini 모델이 URL만으로 웹페이지 내용을 직접 참조 가능
- **최대 20개 링크** 동시 지원 — 최신 정보 기반 그라운딩에 활용

### 💻 AI Studio 코드 에디터
- Gemini 2.5 Pro 탑재 네이티브 코드 에디터 추가
- 자동 오류 감지 및 self-correct 기능
- **Cloud Run 원클릭 배포** 기능 지원

### 🔧 MCP(Model Context Protocol) 네이티브 지원
- **Google Genai SDK**에서 MCP 정의를 네이티브로 지원
- 오픈소스 MCP 툴과의 에이전트 앱 통합 간소화
- Google Maps MCP 연동 데모 시연

### 📱 Android AI / 멀티 폼팩터
- **Androidify**: Gemini 멀티모달 + Imagen3로 셀카 → Android 봇 이미지 생성
- **GenAI API (Gemini Nano)**: 온디바이스 AI — 요약, 재작성, 이미지 설명 로컬 처리
- **Android XR SDK 개발자 프리뷰 2**: Material XR 컴포넌트, 에뮬레이터 업데이트
- 적응형 앱 하나로 **자동차·XR 포함 5억+ 추가 기기** 커버 가능
- **Compose Navigation 라이브러리 전면 재작성**: 화면 스택·상태·애니메이션 개선
- **Android Studio Journeys 에이전트**: 자연어 기반 E2E 테스트 자동화
- **Version Update Agent**: 프로젝트 의존성 자동 업데이트

- **Google Search**가 AI Mode를 통해 전면 재설계되며, 개발자/사용자 경험 모두 근본적으로 변화

---

## 주요 발표/내용

### 🧠 Gemini 2.5 Pro & Deep Think
- LMArena 전 카테고리 석권, 코딩 플랫폼에서 높은 인기
- **Deep Think 모드** 신규 도입: 병렬 추론 기법을 활용한 고급 사고/추론 기능
- **LearnLM** 통합으로 교육 분야 최고 성능 모델로 발전

### ⚡ Gemini 2.5 Flash & Gemini Diffusion
- Flash: 추론, 코드, 롱컨텍스트 등 거의 모든 벤치마크에서 개선, **6월 초 GA 예정**
- **Gemini Diffusion**: 텍스트 확산 모델 기반 실험적 모델, 기존 대비 **5배 빠른 생성 속도**

### 🤖 Agent Mode & Project Mariner
- Gemini 앱에 **Agent Mode** 도입: 아파트 검색 등 복잡한 다단계 작업을 백그라운드에서 자동 처리
- Project Mariner의 에이전트 기능을 **Chrome, Search, Gemini 앱**에 통합
- 티켓 구매 등 실제 트랜잭션까지 처리 가능

### 💬 개인화 스마트 답장 (Personalized Smart Reply)
- Google Drive, Gmail 등 앱 컨텍스트를 활용해 **사용자 말투/스타일/어휘**를 학습한 자동 답장 생성
- 사용자 허가 기반, 투명하게 통제 가능

### 🔍 AI Mode (Google Search 전면 개편)
- 미국 전체 사용자 대상 **오늘부터 순차 롤아웃**
- Gemini 2.5 기반의 복잡한 멀티스텝 쿼리 처리
- **카메라 기반 실시간 검색**, 데이터 시각화(그래프 자동 생성), 쇼핑 개인화 기능 포함
- 패션 특화 커스텀 이미지 생성 모델로 **AI 가상 피팅** 지원

### 📹 Google Beam & 실시간 통역
- 2D 영상을 **3D 영상으로 변환**하는 AI 영상 커뮤니케이션 플랫폼
- Google Meet에 **실시간 음성 번역** 기능 도입

## 핵심 요약 (3줄)
- CSS만으로 스크롤 기반 캐러셀 UI를 구현할 수 있는 새로운 CSS Carousel API가 출시되었다

- Gemini + Firebase 기반의 하이브리드 AI 솔루션으로 서버-클라이언트 AI 협업이 가능해졌다

---

## 주요 발표 내용

### 🎨 UI / CSS
- **CSS Carousel API**: 소량의 CSS 코드만으로 스크롤 기반 네비게이션 및 콘텐츠 정렬 구현 가능
- **Interest Invoker API + Interpositioning API + Popover API 조합**: 인터랙티브 호버 카드(툴팁) 등 리치한 UI 컴포넌트를 선언적으로 구현 가능

### 📏 Web Platform / Base Line
- **Base Line 100% 달성**: 웹 플랫폼 피처 전체를 Base Line으로 매핑 완료
- VS Code, ESLint 등 일상적으로 사용하는 개발 도구에 Base Line 정보가 통합되어 브라우저 호환성 확인이 더 쉬워짐

### 🤖 AI / DevTools
- **Gemini in DevTools**: DevTools 내에서 직접 Gemini에게 질문하여 즉각적인 디버깅 지원 가능
- **Chrome의 Gemini Nano 업그레이드**: 실험적 멀티모달 AI API 추가 (온디바이스 AI)
- **하이브리드 AI 솔루션 (Gemini + Firebase)**: 서버사이드 AI와의 원활한 협업 지원

## 개발자에게 중요한 포인트

- **CSS Carousel API**: 기존 JS 라이브러리(Swiper 등)를 순수 CSS로 대체 → 번들 크기 축소 및 성능 개선
- **Interest Invoker + Popover API**: 접근성(a11y) 고려한 인터랙티브 UI를 마크업 수준에서 처리 → JS 의존도 감소
- **Base Line VS Code/ESLint 통합**: 코드 작성 시점에 브라우저 호환성 경고 → 크로스 브라우저 이슈 사전 예방
- **Gemini in DevTools**: 에러 해석·코드 제안을 DevTools 이탈 없이 처리 → 디버깅 워크플로우 단축
- **온디바이스 + 서버사이드 하이브리드 아키텍처**: 오프라인 지원 및 레이턴시 최적화 전략에 중요한 패턴

## 출시 일정 / 버전 정보
- Gemini Nano 멀티모달 API는 **실험적(Experimental)** 단계
- 자세한 일정은 [Chrome for Developers](https://developer.chrome.com) 채널 확인 권장
