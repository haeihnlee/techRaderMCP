# Google I/O 2025 Developer Keynote 요약

## 핵심 포인트

- **Gemini 2.5 Pro/Flash**가 개발 전반에 깊숙이 통합되어 코딩, 디자인, 테스트 자동화까지 지원
- **멀티모달 Live API**가 공식 출시되어 음성·이미지 기반 대화형 에이전트 개발이 현실화됨
- **AI Studio**가 단순 실험 도구를 넘어 Cloud Run 배포까지 지원하는 풀스택 개발 환경으로 진화
- **Android**는 폴더블·태블릿·자동차·XR까지 하나의 적응형 앱으로 5억+ 기기를 커버하는 방향으로 전환
- **AI 기반 개발 자동화**(테스트 생성, 종속 항목 업데이트 등)로 반복 작업 부담이 크게 감소

---

## 주요 발표/내용

### 🎨 Stitch (실험적 디자인 도구)
- 텍스트 프롬프트 → UI 디자인 + 마크업 코드 자동 생성 (약 1분 소요)
- Gemini 2.5 Pro/Flash 기반, 다크 모드·색상·반경 등 실시간 리믹스 가능
- Figma로 바로 export 가능
- 🔗 `labs.google/stitch`

### 🎙️ Gemini Live API (업데이트)
- **2.5 Flash 네이티브 오디오 모델** 탑재 — 24개 언어 기본 지원
- **Proactive Audio**: 주변 소음/비음성 입력 필터링 개선
- **URL Context 도구**: 최대 20개 URL을 컨텍스트로 직접 활용 가능
- **비동기 함수 호출(non-blocking)** 지원 → MCP 서버 호출 등 지연 작업 중에도 대화 끊김 없음
- 함수 호출 시 **구조화된 JSON 출력** 개선

### 🛠️ Google AI Studio 업데이트
- **내장 코드 편집기**에 Gemini 2.5 Pro 통합 → 웹앱 생성·오류 자동 수정
- **Cloud Run 원클릭 배포** 기능 추가
- 멀티턴 대화 기반 반복 개발 환경 지원
- **MCP(Model Context Protocol)** 정의를 Google Gen AI SDK에서 기본 지원

### 📱 Android AI 개발
- **Androidify 샘플 앱**: Gemini 멀티모달 + Imagen3로 셀카 → Android 봇 이미지 생성 (코드 5줄 수준)
- **GenAI API (Gemini Nano 기반)**: 온디바이스 AI — 요약, 재작성, 이미지 설명 등 로컬 처리
- **Firebase를 통한 클라우드 AI** 연동 간소화

### 💅 Material 3 Expressive
- 새로운 도형 라이브러리, 부드러운 버튼 전환 애니메이션 등 UI 표현력 강화
- Compose Material 알파에서 즉시 사용 가능

### 📐 Android 적응형 앱 / 멀티 폼팩터
- **Android 16**: 방향 전환·크기 조정·가로세로 비율 제한 API 개선
- **실시간 업데이트(Live Updates)**: 내비게이션·배달·차량 공유 등 시간 민감 정보 알림 표시
- **삼성 DeX** 지원 강화, 데스크톱 창 모드 개선
- **Android XR SDK 개발자 프리뷰 2** 출시: Material XR 컴포넌트, 에뮬레이터 업데이트, 공간 동영상 지원
- XREAL의 **Aura 프로젝트** 기기 개발자 버전 곧 출시 예정
- 적응형 앱 하나로 **자동차·XR 포함 5억+ 추가 기기** 커버 가능

### ⚡ Jetpack Compose / Android Studio 업데이트
- **Compose Navigation 라이브러리 전면 재작성**: 화면 스택·상태·애니메이션·적응형 레이아웃 개선
- **CameraX·Media3 Compose 라이브러리** 출시
- 최신 Compose 버전에서 자동 완성·텍스트 자동 크기 조절·공개 상태 추적 기능 추가
- 구형 Pixel 기기에서도 저품질 프레임(jank) 대폭 감소

### 🤖 Android Studio AI 에이전트 기능
- **자연어 기반 E2E 테스트 생성**: 자연어로 사용

## 핵심 포인트

- AI가 단순 정보 검색을 넘어 **에이전트(Agent) 방식**으로 실제 작업을 대신 처리하는 시대로 전환
- **Google Search**가 AI Mode를 통해 전면 재설계되며, 개발자/사용자 경험 모두 근본적으로 변화

---

## 주요 발표/내용

### 🧠 Gemini 2.5 Pro & Deep Think
- LMArena 전 카테고리 석권, 코딩 플랫폼에서 높은 인기
- **Deep Think 모드** 신규 도입: 병렬 추론 기법을 활용한 고급 사고/추론 기능
- **LearnLM** 통합으로 교육 분야 최고 성능 모델로 발전

### ⚡ Gemini 2.5 Flash & Gemini Diffusion
- Flash: 추론, 코드, 롱컨텍스트 등 거의 모든 벤치마크에서 개선, **6월 초 GA 예정**
- **Gemini Diffusion**: 텍스트 확산 모델 기반 실험적 모델, 기존 대비 **5배 빠른 생성 속도**

### 🤖 Agent Mode & Project Mariner
- Gemini 앱에 **Agent Mode** 도입: 아파트 검색 등 복잡한 다단계 작업을 백그라운드에서 자동 처리
- Project Mariner의 에이전트 기능을 **Chrome, Search, Gemini 앱**에 통합
- 티켓 구매 등 실제 트랜잭션까지 처리 가능

### 💬 개인화 스마트 답장 (Personalized Smart Reply)
- Google Drive, Gmail 등 앱 컨텍스트를 활용해 **사용자 말투/스타일/어휘**를 학습한 자동 답장 생성
- 사용자 허가 기반, 투명하게 통제 가능

### 🔍 AI Mode (Google Search 전면 개편)
- 미국 전체 사용자 대상 **오늘부터 순차 롤아웃**
- Gemini 2.5 기반의 복잡한 멀티스텝 쿼리 처리
- **카메라 기반 실시간 검색**, 데이터 시각화(그래프 자동 생성), 쇼핑 개인화 기능 포함
- 패션 특화 커스텀 이미지 생성 모델로 **AI 가상 피팅** 지원

### 📹 Google Beam & 실시간 통역
- 2D 영상을 **3D 영상으로 변환**하는 AI 영상 커뮤니케이션 플랫폼
- Google Meet에 **실시간 음성 번역** 기능 도입
